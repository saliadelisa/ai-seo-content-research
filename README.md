# AI-Powered SEO Content Production — Research Repository

## Overview

This repository documents a structured research project on **AI-powered SEO content production** — how practitioners are using AI tools (LLMs, automation pipelines, semantic analysis) to scale content output without sacrificing quality.

The goal is to collect high-signal insights from people who **actually build and run** AI-assisted SEO workflows, not just write about them theoretically. This research will serve as the foundation for a practical playbook.

---

## Why This Topic

AI-powered SEO content production sits at the intersection of:
- **Technical implementation** (APIs, automation, prompt engineering)
- **Business outcomes** (organic traffic, lead generation, content ROI)
- **Strategic thinking** (content architecture, topical authority, quality control)

---

## Expert Selection Criteria

Experts were selected based on:
1. **Active practitioners** — people running real SEO operations, not just consultants writing blog posts
2. **Documented AI experimentation** — publicly shared workflows, results, or tools
3. **Platform diversity** — covering LinkedIn, YouTube, newsletters, and podcasts
4. **Perspective diversity** — skeptics, optimists, and tool builders all represented

## Collection Method

- **LinkedIn posts**: Manually collected from each expert's profile (latest 3–5 posts relevant to AI + SEO)
- **YouTube transcripts**: Collected via Supadata API using Python script (`scripts/fetch_transcripts.py`)
- **Other content**: Newsletter archives and podcast show notes collected manually
